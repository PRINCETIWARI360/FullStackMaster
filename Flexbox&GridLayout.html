Flexbox & Grid Layouts

Modern CSS Layout Techniques

CSS Flexbox Layout

Flexbox is a one-dimensional layout method for arranging items in rows or columns. It provides efficient ways to align, distribute, and reorder elements within a container.

Flex Container Properties

flex-direction

Defines the direction of flex items (row, row-reverse, column, column-reverse)

.container { flex-direction: row; }

flex-wrap

Controls whether items wrap or not (nowrap, wrap, wrap-reverse)

.container { flex-wrap: wrap; }

justify-content

Aligns items along the main axis (flex-start, center, flex-end, space-between, space-around)

.container { justify-content: center; }

align-items

Aligns items along the cross axis (stretch, flex-start, center, flex-end, baseline)

.container { align-items: center; }

Flex Item Properties

flex-grow

Defines how much an item can grow relative to others

.item { flex-grow: 1; }

flex-shrink

Defines how much an item can shrink relative to others

.item { flex-shrink: 0; }

flex-basis

Defines the initial size of an item before space distribution

.item { flex-basis: 200px; }

order

Controls the order in which items appear

.item { order: 2; }

CSS Grid Layout

CSS Grid is a two-dimensional layout system for creating complex responsive web designs. It works by dividing a page into major regions.

Grid Container Properties

grid-template-columns

Defines the columns of the grid (e.g., 1fr 1fr 1fr)

.container { grid-template-columns: 1fr 1fr 1fr; }

grid-template-rows

Defines the rows of the grid (e.g., 100px auto 200px)

.container { grid-template-rows: 100px auto 200px; }

grid-gap / gap

Defines the gap between grid items (e.g., 20px)

.container { gap: 20px; }

grid-template-areas

Defines a grid template by referencing named areas

.container { grid-template-areas: "header header" "sidebar main" "footer footer"; }

Grid Item Properties

grid-column

Defines an item's position within the grid columns

.item { grid-column: 1 / 3; }

grid-row

Defines an item's position within the grid rows

.item { grid-row: 2 / 4; }

grid-area

Assigns an item to a named area or defines its position

.item { grid-area: header; }

justify-self & align-self

Aligns a single item along the row/column axis

.item { justify-self: center; align-self: center; }

Flexbox vs Grid

Flexbox and Grid are both powerful layout systems in CSS, but they have different strengths and use cases.

When to Use Each

Use Flexbox When:

  • You need a one-dimensional layout (either row or column)
  • You need content-first design (items size themselves)
  • You need alignment control within a single axis
  • For navigation bars, form controls, and small UI components

Use Grid When:

  • You need a two-dimensional layout (rows and columns)
  • You need precise control over both rows and columns
  • You need to create complex layouts with overlapping areas
  • For page layouts, dashboard grids, and image galleries

Feature Comparison

Feature Flexbox Grid
Dimensions One-dimensional (row OR column) Two-dimensional (rows AND columns)
Content Flow Content-based (items size themselves) Container-based (container defines structure)
Alignment Strong alignment capabilities on one axis Alignment on both axes simultaneously
Item Control Items can be reordered with order property Items can be placed anywhere in the grid
Overlap Items cannot overlap Items can overlap and layer with z-index
Use Cases Navigation bars, form controls, small components Page layouts, complex dashboards, image galleries

Pro Tips

  • You can and should use Flexbox and Grid together in the same project
  • Use Grid for overall page layout and Flexbox for individual components
  • Grid is ideal for creating responsive layouts without media queries
  • Flexbox is perfect for centering content both vertically and horizontally
  • Both systems have excellent browser support in modern browsers
