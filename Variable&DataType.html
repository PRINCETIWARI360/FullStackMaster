Variables in JavaScript

Variables are containers for storing data values. JavaScript has three ways to declare variables: var, let, and const.

Variable Declaration

// Using var (function-scoped)
var name = "John Doe";

// Using let (block-scoped)
let age = 30;

// Using const (block-scoped, constant)
const PI = 3.14159;

Variable Reassignment

let counter = 0;
counter = 1; // Valid reassignment

const MAX_USERS = 100;

Data Types in JavaScript

JavaScript has dynamic types. The same variable can hold different data types. There are 7 primitive data types and 1 complex type.

Primitive Data Types

String

Represents textual data

let text = "Hello World";

Number

Represents numeric values

let num = 42;

Boolean

Represents true/false values

let isActive = true;

Null

Represents intentional absence of value

let empty = null;

Undefined

Represents uninitialized variables

let notDefined;

Symbol

Represents unique identifiers

const id = Symbol("id");

BigInt

Represents large integers

const bigNum = 12345678901234567890n;

Complex Data Type

Object

Represents collections of key-value pairs

let person = {
  name: "John",
  age: 30
};

Type Conversion & Checking

Type Conversion

JavaScript can convert between types automatically (coercion) or manually.

// String to Number
let num = Number("42"); // 42

// Number to String
let str = 42.toString(); // "42"

// Boolean Conversion
let bool = Boolean(1); // true

Type Checking

Use typeof operator to check the type of a variable.

typeof "Hello"; // "string"
typeof 42; // "number"
typeof true; // "boolean"
typeof {}; // "object"
typeof undefined; // "undefined"
typeof null; // "object" (historical quirk)
