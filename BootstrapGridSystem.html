Bootstrap Grid System

Master responsive layouts with Bootstrap's powerful grid system

Introduction to Bootstrap Grid

Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page. The grid system uses a series of containers, rows, and columns to layout and align content.

Key Features

  • Responsive by design
  • 12-column grid layout
  • Five responsive tiers (xs, sm, md, lg, xl)
  • Flexbox-based for better alignment
  • Mobile-first approach
  • Easy nesting of columns

Grid Structure

Bootstrap's grid consists of 12 equally sized columns. You can combine these columns to create different layouts.

Responsive Breakpoints

Bootstrap includes five responsive breakpoints based on minimum viewport widths. These determine when your layout will adapt to accommodate different screen sizes.

Extra Small (xs)

<576px
Default for mobile devices

Small (sm)

≥576px
Small tablets and large phones

Medium (md)

≥768px
Tablets in landscape mode

Large (lg)

≥992px
Small desktops and laptops

Extra Large (xl)

≥1200px
Large desktops

Advanced Grid Features

Column Offsetting

Move columns to the right using offset classes.

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
.col-md-3 .offset-md-3

Column Reordering

Change the visual order of your content.
