CSS Transitions

CSS transitions allow you to change property values smoothly over a given duration. They provide a way to control animation speed when changing CSS properties.

Transition Properties

transition-property

Specifies the name of the CSS property the transition effect is for

.box { transition-property: all; }

transition-duration

Specifies how many seconds or milliseconds a transition effect takes to complete

.box { transition-duration: 0.5s; }

transition-timing-function

Specifies the speed curve of the transition effect

.box { transition-timing-function: ease-in-out; }

transition-delay

Specifies a delay (in seconds) for the transition effect

.box { transition-delay: 0.2s; }

CSS Animations

CSS animations make it possible to animate transitions from one CSS style configuration to another. Animations consist of two components: a style describing the CSS animation and a set of keyframes that indicate the start and end states of the animation's style.

Animation Properties

animation-name

Specifies the name of the keyframe animation

.box { animation-name: slide; }

animation-duration

Specifies how long an animation should take to complete one cycle

.box { animation-duration: 2s; }

animation-iteration-count

Specifies the number of times an animation should be played

.box { animation-iteration-count: infinite; }

animation-direction

Specifies whether an animation should play forwards, backwards, or alternately

.box { animation-direction: alternate; }

animation-timing-function

Specifies the speed curve of the animation

.box { animation-timing-function: ease-in-out; }

animation-delay

Specifies a delay before the animation will start

.box { animation-delay: 1s; }

animation-fill-mode

Specifies what values are applied before/after the animation

.box { animation-fill-mode: forwards; }

animation-play-state

Specifies whether the animation is running or paused

.box { animation-play-state: paused; }
