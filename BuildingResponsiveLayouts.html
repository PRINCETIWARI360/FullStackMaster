Building Responsive Layouts

Design for All Devices

Responsive Design Fundamentals

Responsive web design is an approach that makes web pages render well on a variety of devices and window or screen sizes. The goal is to build a single website that adapts to provide an optimal viewing experience across different devices.

Fluid Layouts

Use relative units like percentages instead of fixed units like pixels for layout elements.

.container { width: 90%; max-width: 1200px; margin: 0 auto; }

Flexible Images

Ensure images scale properly to fit their containers without distortion.

img { max-width: 100%; height: auto; }

Relative Units

Use rem and em for font sizes and spacing to ensure proportional scaling.

body { font-size: 1rem; }
h1 { font-size: 2.5rem; }

Viewport Meta Tag

Essential for responsive design on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Responsive Units Explained

Resize me!
.box { width: 200px; height: 100px; }

Media Queries

Media queries allow you to apply CSS styles depending on device characteristics such as width, height, or orientation. They are the foundation of responsive design.

Breakpoints & Syntax

Mobile First

Start with mobile styles and enhance for larger screens.

/* Base styles (mobile) */
.container { padding: 10px; }

/* Tablet styles */ @media (min-width: 768px) { .container { padding: 20px; } }

/* Desktop styles */ @media (min-width: 1024px) { .container { padding: 40px; } }

Common Breakpoints

  • Mobile: < 576px
  • Tablet: 576px - 768px
  • Small Desktop: 768px - 992px
  • Desktop: 992px - 1200px
  • Large Desktop: > 1200px

These are guidelines only. Always base breakpoints on your content.

Flexbox Layouts

Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex to fill additional space or shrink to fit into smaller spaces.

Flexbox Properties

Flex Containers

Properties applied to the parent element (flex container):

  • display: flex
  • flex-direction
  • flex-wrap
  • justify-content
  • align-items

Flex Items

Properties applied to the children (flex items):

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • align-self

CSS Grid Layout

CSS Grid Layout excels at dividing a page into major regions or defining the relationship in terms of size, position, and layer between parts of a control built from HTML primitives.

Grid Terminology

Grid Container

The element on which display: grid is applied. It's the direct parent of all the grid items.

.container { display: grid; }

Grid Lines

The dividing lines that make up the structure of the grid. They can be vertical ("column grid lines") or horizontal ("row grid lines").

Grid Tracks

The space between two adjacent grid lines. You can think of them as the columns or rows of the grid.

Grid Areas

The rectangular space surrounded by four grid lines. A grid area can contain any number of grid cells.
