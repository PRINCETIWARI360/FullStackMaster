Understanding HTML Structure

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and content of web pages.

Basic HTML Structure

Every HTML document contains these fundamental elements:

  • DOCTYPE Declaration - Defines the document type and version
  • HTML Element - The root element of the document
  • Head Section - Contains meta information about the document
  • Body Section - Contains the visible page content

Key HTML Elements

  • <title> - Defines the title shown in browser tabs
  • <meta> - Provides metadata about the HTML document
  • <link> - Used to link external resources like CSS
  • <script> - Embeds or references JavaScript code
  • <div> - A container for other HTML elements
  • <p> - Defines a paragraph of text
  • <a> - Creates hyperlinks to other pages
  • <img> - Embeds images in the document

HTML Structure Example

<!DOCTYPE html> <html lang="en"> <head>   <meta charset="UTF-8">   <meta name="viewport" content="width=device-width, initial-scale=1.0">   <title>Document Title</title>   <link rel="stylesheet" href="styles.css"> </head> <body>   <header>     <h1>Welcome to My Website</h1>   </header>   <nav>     <ul>       <li><a href="#">Home</a></li>       <li><a href="#">About</a></li>       <li><a href="#">Contact</a></li>     </ul>   </nav>   <main>     <article>       <h2>Article Title</h2>       <p>This is a paragraph of text in the article.</p>     </article>   </main>   <footer>     <p>© 2023 My Website</p>   </footer>   <script src="script.js"></script> </body> </html>

Document Structure Flow

  1. DOCTYPE Declaration - Specifies HTML5 document
  2. HTML Root Element - Wraps entire document
  3. Head Section - Contains meta data and links
  4. Body Section - Contains all visible content
  5. Content Sections - Organized with semantic tags
  6. Scripts - JavaScript at the end for performance
